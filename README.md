# Python-Programming-BlackJack
Simulatore funzionante del BlackJack

## Descrizione
Il progetto consiste in una simulazione interattiva del BlackJack sviluppata per approfondire la gestione logica decisionale e dei flussi di gioco. L'applicativo sfida l'utente a competere contro il banco, implementando le regole classice e gestendo in tempo reale il calcolo del punteggio.

## Funzionalità principali
**Validazione input ed etica:** Implementazione di un modulo di verifica dell'età per limitare l'accesso al gioco ai soli utenti maggiorenni, dimostrando attenzione alla gestione dei requisiti di sistema e alla validazione dei dati utente. 
**Logica decisionale:** Gestione dei turni del giocatore (Carta? sì/no). 
**AI del banco:** Algoritmo automatico per le giocate del banco basato su punteggi standard. 
**Controllo punteggio:** Verifica automatica del superamento del 21 (BlackJack) e confronto dei risultati.

## Tecnologie utilizzate
**Linguaggio:** Python. 
**Ambiente:** Visual Studio Code.
